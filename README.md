# Mini Shell in C

A custom command-line interpreter built using C and Win32 APIs.

## Features
- Custom prompt
- Built-in commands
- Command parsing
- Current directory display
- File listing
- External command execution

## Technologies
- C
- Win32 API
- Visual Studio

## Build

cl main.c shell.c commands.c
